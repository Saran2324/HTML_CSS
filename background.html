Road-safety

This project proposes an automatic accident data collection system to reduce accident rates in the country. The system collects accident data when detected, identifying hotspots, and implementing necessary technology. It uses Arduino for message transfer and GPS to locate accident spots. The system notifies nearest medical centers and registered mobile numbers about the accident location, using GSM and GPS modules. A vibration sensor detects accidents, and the system can extend to providing medication to victims at the accident spot. Additionally, technology can be used to prevent accidents by implementing alert systems.
